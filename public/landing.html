API Endpoints

RESTful by design

All endpoints are authenticated via X-API-Key header.

GET/healthzHealth check
GET/api/statusWhatsApp status
GET/api/qrQR code
POST/api/send-messageSend text
POST/api/send-imageSend image
POST/api/send-documentSend document
POST/api/send-audioSend audio
POST/api/send-locationSend location
GET/api/messagesMessage history
GET/api/docsSwagger UI

Send with curl

# Send a text message
curl -X POST http://your-server/api/send-message \
  -H "X-API-Key: wapi_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{"to":"923001234567","message":"Hello from the API!"}'
# Send an image
curl -X POST http://your-server/api/send-image \
  -H "X-API-Key: wapi_your_api_key_here" \
  -F "to=923001234567" \
  -F "file=@photo.jpg" \
  -F "caption=Check this out!"
